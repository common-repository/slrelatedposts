== Description ==

Liersch Related Posts is a little plug-in for WordPress. It contains an easy to use widget to show related posts on the sidebar. Additionally, a content filter can be installed to list related posts at the end of each post.

If you have any questions or if you need more information, please visit [http://www.steffen-liersch.de/wordpress/](http://www.steffen-liersch.de/wordpress/ "Steffen Liersch").

== Installation ==

1. Upload the plug-in folder to the `/wp-content/plugins/` directory
2. Activate the plug-in through the 'Plugins' menu in WordPress
3. Add the widget to your sidebar or use the content filter
